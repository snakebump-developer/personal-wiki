# Per vedere la lista dei commit con il terminale

Per farlo si usa il comando dentro la cartella clonata:

```bash
git log --oneline
```

Se sei bloccato nella visualizzazione di git log all'interno del terminale, significa che il log è stato aperto in un programma chiamato less, che è il visualizzatore di default per i file lunghi o gli output estesi.

Per uscire e tornare al normale prompt del terminale, premi semplicemente il tasto `q` (come "quit") sulla tastiera.
Ecco i passaggi esatti:

Assicurati che il terminale sia attivo (cliccaci sopra).

Premi il tasto `q`.

Dovresti immediatamente uscire dalla visualizzazione del log e vedere di nuovo il cursore lampeggiare accanto al prompt dei comandi ($, %, ecc.).

Mentre sei in quella visualizzazione (dentro less), puoi anche usare altri comandi utili:

- Invio o j: Scorri giù di una riga.
- Spazio o f: Scorri giù di una schermata intera.
- b: Scorri su di una schermata intera.
- /: Cerca un testo specifico nel log.
