# Come fare deploy su vercel con applicazione Astro

Se il progetto è stato trasformato in SSR come prima cosa serve lanciare il comando per installare un Adapter di vercel

`npx astro add vercel`

Dopo completare gli ultimi push 
Poi andiamo su internet e entriamo nella dashboard del sito web di vercel con il profilo github legato alla repo
Nella lista di repo dell'account che vediamo direttamente nella dashboard selezioniamo la repo che vogliamo deploiare e poi clicchiamo su Deploy
Attendiamo che completi le operazioni Vercel ed è tutto ora siamo ingrado di visualizzare il nostro progetto realizzato interamente con solo Astro + Vercel
