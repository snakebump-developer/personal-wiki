# Come creare un progetto con Astro

1. Lanciamo il comando nel terminale nella cartella progetto designata da noi:

`npm create astro@latest`

2. Compiliamo le domande con:

    - Dove eseguiamo la creazione: `.`
    - Struttura cartelle: `empty`
    - Installare le dipendenze: `yes`
    - Creare una repository: `no`

3. Lanciamo al termine il comando per attivare il server:

`npm run dev`
